## Setup
#### Clone this repository
```bash
git clone https://github.com/SaranshDawra/Pomodoro-Clock.git
```
#### Move to desired directory
```bash
cd Pomodoro-Clock
```
#### To install the dependencies
```bash
npm install
```
#### To run the app
```bash
npm start
```
